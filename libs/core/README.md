## Core Libraries

version=1.0.0

### Библиотеки в составе ядра

- **Api Client Lib** (`apiclient`): Универсальный API-клиент с поддержкой обработки ошибок, повторных запросов и ограничений по скорости.
- **Logger** (`logger`): Библиотека для логирования с возможностью отправки логов в RabbitMQ.
- **Performance Profiler** (`perfprofiler`): Библиотека для профилирования функций и корутин, с возможностью мониторинга времени выполнения и потребления памяти. 
- **Database** (`database`): Универсальная библиотека для типичных запросов работы с БД.
- **Smart Kafka** (`tradeforge_kafka`): Библиотека для умной обработки ошибок, исключений и работы с Kafka.
- **TradeForge Schemas** (`tradeforge_schemas`): Универсальный набор pydantic схем, необходимый для всего проекта.

Эти библиотеки обеспечивают базовую функциональность и предназначены для использования в других сервисах ParserHub.
