# Trade Forge: Internal API

## Назначение

Этот сервис является **внутренним API (Internal API)** платформы "Trade Forge". Он выполняет роль центрального оркестратора бизнес-логики.

### Ключевые ответственности:

-   Прием и валидация запросов от внешнего API (Gateway).
-   Управление пользовательскими сущностями (стратегии, бэктесты) в PostgreSQL.
-   Инициация асинхронных, ресурсоемких задач (например, бэктестинг) путем отправки сообщений в Kafka.
-   Предоставление справочной информации (индикаторы, тикеры) для UI.
-   Контроль пользовательских квот и лимитов.

Этот сервис **не выполняет** тяжелых вычислений, таких как расчет индикаторов или симуляция бэктестов. Он только делегирует эти задачи специализированным воркерам через брокер сообщений.
